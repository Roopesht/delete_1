## Visitor (Anonymous) Site

### Marketing Information
- Overview of The Trade Tribe
- Value proposition and learning approach
- Learning path (Free + Premium overview)
- Trust and credibility (trainer background)
- Clear CTA to register
- Simple message to be delivered (Join  -> Learn  -> Observe setups  -> Practice with guidance  -> Build confidence  -> Trade independently)

### Registration
- User registers with:
  - Name
  - Phone number
  - Email
  - Aspiration (why I joined this program)
- After registration:
  - User is onboarded into the system
  - Confirmation is shown
  - Next steps shared via **WhatsApp** (program updates, enrollment guidance)
  - Invited to join **Telegram** community (trading content, education, setups)

## Registered User Section

### Access & Experience
- Access to Telegram community (handled externally)

  - Enrolled in F&O
